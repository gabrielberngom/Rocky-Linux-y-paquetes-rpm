# Operaciónes Basicas

- Actualización de la lista de paquetes.

no es necesarrio, actualizar la lista antes de actualizar

- Actualización de paquetes instalados.

yum -y update

- Búscar un paquete.

yum search

- Averiguar si un paquete está instalado.

yum list --installed

- Instalar un paquete.

yum install

- Borrar un paquete.

yum remove
